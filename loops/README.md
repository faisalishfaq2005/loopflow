# The Loop Cookbook

Community-contributed loops. Copy any file into your project's `.loopflow/loops/` and adapt the prompts.

To add yours, see [CONTRIBUTING.md](../CONTRIBUTING.md) — the bar is simple: it solved a real problem for you, it's generalized, and it validates.

| Loop | What it does |
|---|---|
| [`pr-ready.yaml`](pr-ready.yaml) | Polishes a feature branch until it would survive review: lint, tests, self-review gate |
